## Recent Projects
Here are 4 projects I've completed in my most recent semesters from the courses:  Machine Learning, Predictive Analytics, Text Mining and Social Media Analytics. These projects together demonstrate my ability to create data analytics pipelines to collect,clean and transform data to build and evaluate models using different machine learning techniques in Python. Here is an aggregation of skills developed during the projects below:

* **Engineered end-to-end data pipelines** by collecting and generating datasets from RESTful APIs (YouTube V3, GitHub) and structured CSV sources; cleaned, imputed, transformed, and validated data using pandas and NumPy to ensure high-quality inputs for EDA and model training — resulting in reproducible, analysis-ready datasets.
* **Explored Data (EDA)** by analyzing distributions, correlations, and feature importance using statistical summaries, visualizations with Matplotlib(PyPlot/Leafletjs), and clear markdown documentation. Uncovered trends and data issues (missing data, leaky labels/variables) leading to insights for feature selection, engineering and model interpretation/performance.
* **Applied structured model selection** and justification by comparing multiple algorithms (e.g., Logistic Regression, XGBoost (Tree classifier), LSTM, Clustering Models) using cross-validation and performance metrics; selected models based on bias–variance tradeoffs and domain considerations leading to better model performance and strong foundational stastitical interpretation of results.
* **Built, tuned, and evaluated predictive models** by implementing hyperparameter tuning with GridSearchCV and  randomized search, feature engineering, as well as train-test set size optimization. Evaluated performance using metrics where appropriate such as Accuracy, F1-score, RMSE, Silhouette score; resulting in a better understanding on how to further optimize models.
* **Communicated insights effectively** by summarizing findings through visualizations, markdown documentation, and performance comparisons; turning complex statistical results into interpretable conclusions/laymans terms.
